🔗 Comandos de Instalación

Ejecuta estos comandos en orden:

1. Actualizar base de datos: 
mysql -u root -p newapp < database/migrate_users.sql
2. Migrar imágenes de usuarios: 
php database/migrate_profile_images.php
3. O desde el navegador: 
http://localhost/Nueva-WEB/database/migrate_profile_images.php

🧪 Pruebas Recomendadas

  1. Registro sin imagen: Debe copiar avatar según género
  2. Registro con imagen: Debe guardar imagen en uploads/users/{ID}/
  3. Login: Debe mostrar imagen de perfil en menú usuario
  4. Actualización de perfil: Debe permitir cambiar imagen
  5. Persistencia tras reload: Menú usuario debe mantenerse visible
